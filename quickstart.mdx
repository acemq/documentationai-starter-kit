## Prerequisites

Begin your AceMQ engagement by reaching out through the preferred channel. AceMQ specializes in RabbitMQ commercial licensing, support, and consulting for message queuing solutions.

<Steps>
  <Step title="Visit Contact Page" icon="external-link">

  </Step>

  <Step title="Provide Environment Details" icon="database">

  </Step>
</Steps>

## Request a Free MQ Assessment

AceMQ offers free assessments for your message queuing environment. Choose the broker type.

<Tabs>
  <Tab title="RabbitMQ" icon="rabbit">
    Request a RabbitMQ-specific review.
    
    ```bash
    curl -X POST https://acemq.com/api/assess \
      -H "Content-Type: application/json" \
      -d '{
        "broker": "RabbitMQ",
        "version": "3.12",
        "issues": "High latency in queues"
      }'
    ```
  </Tab>
  
  <Tab title="Redis" icon="database">


## Obtain RabbitMQ Commercial Licensing

For production RabbitMQ deployments, secure commercial licensing.

<CodeGroup tabs="cURL,Python">
```bash
curl -X POST https://acemq.com/licensing/request \
  -H "Content-Type: application/json" \
  -d '{
    "company": "Your Company",
    "nodes": 10,
    "support_level": "enterprise"
  }'
```

```python
import requests

response = requests.post(
    "https://acemq.com/licensing/request",
    json={
        "company": "Your Company",
        "nodes": 10,
        "support_level": "enterprise"
    }
)
print(response.json())
```
</CodeGroup>
